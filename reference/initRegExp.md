# Add regular expression operator

Forwarded to `initExtension(db, "regexp")` .

## Usage

``` r
initRegExp(db)
```

## Arguments

- db:

  A
  [`SQLiteConnection`](https://rsqlite.r-dbi.org/reference/SQLiteConnection-class.md)
  object to add the regular expression operator into the connection.

## Value

Always `TRUE`, invisibly.
